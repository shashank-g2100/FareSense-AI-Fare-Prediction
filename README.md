# FareSense AI – Uber Fare Prediction System

FareSense AI is a machine learning based fare prediction system that estimates Uber ride prices using historical ride data.

The project demonstrates a complete ML workflow from data preprocessing to deployment using Streamlit.

---

## Project Overview

This project builds a regression model to predict ride fare based on:

• Distance  
• Passenger count  
• Pickup date and time  
• Ride patterns  

The final model uses **Random Forest Regression** with hyperparameter tuning.

---

## Model Performance

Best Model: Random Forest Regression

Performance:

R² Score: 0.81  
MAE: ~$2  
RMSE: ~4.2  

The model performs best within typical urban ride ranges (0–60 km).
